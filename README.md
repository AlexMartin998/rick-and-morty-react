### Running the app

#### Dev

```bash
npm i -g pnpm

pnpm i

pnpm run dev
```

#### Prod

```bash
# docker compose:
docker compose up --build -d

# docker:
docker build -t hoobak:1.0.0 .

docker run -p 8080:80 hoobak:1.0.0
```
